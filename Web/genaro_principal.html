Página Genara

Bienvenido a la página de GENaro, ahora mismo estamos contruyendola. Para amenizarte la espera puedes ir bajandote la documentación del proyecto, o unas bonitas melodías de GENaro.

Algunos enlaces interesantes:

  • http://audacity.sourceforge.net/ - esto es un editor de wabs y otros formatos con un lenguaje para crear nuevos plugins
  • http://www.math.uga.edu/~djb/html/math-music.html - esta es la página de donde se puede bajar el libro de música y matematicas
  • http://www.iua.upf.es/mtg/clam/ - esto es un motor de audio hecho en la Universidad Pompeu Fabra
  • http://www.hitsquad.com/smm/programs/JammerProf_win32/ - Este es un generador de acompañamientos pseudo-aleatorio
  • http://www.haskell.org/haskore/ - Esta es la página oficial de una biblioteca de haskell para expresar estructuras musicales
  • http://timidity.sourceforge.net/ - esto es un sintetizador software
  • http://nosuch.com/tjt/index.html - Esta es la pagina de Tim Thompson, creador de keykit, vienen ejemplos de música generada de forma aleatoria
  • http://lilypond.org/web/index.html - lilypond es un editor de partituras que acepta midi como entrada asi como un lenguaje propio. Puede generar salida en formatos .ps y .pdf
  • http://babel.dacya.ucm.es/jaime/pru/ - esta es la página de Jaime, viene un ejemplo de la diferencia entre dos sistemas de afinación
